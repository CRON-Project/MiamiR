Fake_COVARSR Documentation

5,000 Participant Simulated Sample of GWAS-like Covariate information

Description

A dataset containing simulated quantitative, binary and categorical covariates for participants The variables are as follows:

Usage

data(Fake_COVARS)

Format

A data frame with 5,000 rows and 8 variables (simulated)

Details

  • FID = Participant Family Identifier

  • IID = Participant Individual Identifier

  • Age = Years of Age

  • Sex = Genetic Sex

  • Location = Sample Region

  • BMI = Continous body mass index

  • PC1 = First Genetic Principle Component

  • PC2 = Second Genetic Principle Component
