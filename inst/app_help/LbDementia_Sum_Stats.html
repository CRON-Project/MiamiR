LbDementia_Sum_Stats

LbDementia_Sum_StatsR Documentation

100,000 SNP Sample Subset of GWAS Summary Statistics of Lewy Body Dementia

Description

A dataset containing a subset of the summary statistics from the Chia, R. 2021 Lewy Body Dementia GWAS The variables are as follows:

Usage

data(LbDementia_Sum_Stats)

Format

A data frame with 100,000 rows (randomly selected) and 10 variables from: https://ftp.ebi.ac.uk/pub/databases/gwas/summary_statistics/GCST90001001-GCST90002000/GCST90001390/

Details

  • chromosome = chromosome number

  • variant_id = SNP rs number

  • effect_allele_frequency = observed frequency of effect allele

  • base_pair_location = basepair position (GRCh38)

  • other_allele = The non-effect allele

  • effect_allele = The effect allele

  • beta = The regression weight

  • odds_ratio = The effect allele test statistic from a logistic regression model

  • standard_error = The standard error of the Beta

  • p_value = The P value of the association test.
