METASOFT_File_GenR Documentation

Munged summary statistics in preparation for METASOFT-like meta-analysis

Description

Munged summary statistics in preparation for METASOFT-like meta-analysis

Usage

METASOFT_File_Gen(..., session = NULL, Debug = FALSE)

Arguments

Match_Allele_Study

Data set used as reference point for effect allele direction; defaults to NULL - automatically assigned as first set

Output

A selection of columns to isolate, in order, alternating by dataset; defaults to NULL, but automatically assigned c("BETA", "SE") upon function call - standard METASOFT ordered format; GENPOS, P, CHROM, ALLELE0, ALLELE1 (all options)

Data

These are the summary statistics (either file path(s), environmental object(s) or a variable(s) containing such contents) to be plotted; defaults to NULL (must be one list only)

BETA_Column

Manually specify BETA Test Statistic column; defaults to NULL, leading to auto-detection

Position_Column

Manually specify chromosome genomic position column; defaults to NULL, leading to auto-detection

SNP_ID_Column

Manually specify SNP ID column; defaults to NULL, leading to auto-detection

PValue_Column

Manually specify P Value column; defaults to NULL, leading to auto-detection

Reference_Allele_Column

Manually specify reference allele column; defaults to NULL, leading to auto-detection

Effect_Allele_Column

Manually specify effect allele column; defaults to NULL, leading to auto-detection

SE_Column

Manually specify SE of Test Statistic column; defaults to NULL, leading to auto-detection

Chromosome_Column

Manually specify chromosome column; defaults to NULL, leading to auto-detection

Output_Reference_Identifier

Include the supplied internal identifier of the Match_Allele_Study dataset in final output; defaults to TRUE

Verbose

Prevent display of progress bar as function is running and instead show key milestone outputs/messages (mainly for debugging purposes); defaults to FALSE

Value

Munged dataframe amenable to METASOFT meta-analysis - only including SNPs present in backbone/reference file as other SNPs effects cannot be consistently aligned

Examples

METASOFT <- METASOFT_File_Gen(Data = c("Intelligence_Sum_Stats", "Household_Income_Sum_Stats"))
