METASOFT Wrapper

Description

METASOFT Wrapper

Usage

METASOFT_Run(..., session = NULL)

Arguments

Data

These are the harmonised summary statistics (either file path(s), environmental object(s) or a variable(s) containing such contents) to be run through the METASFOT wrapper. These must match the METASFOT input file format and can be generated with METASOFT_File_Gen(); defaults to NULL

options

Character vector of extra METASOFT options (e.g. c("-verbose", "mvalue"); defaults to character()/NULL

REGENIE_Cols

Include REGENIE format-style columns in output; defaults to TRUE

Verbose

Prevent display of progress bar as function is running and instead show key milestone outputs/messages (mainly for debugging purposes); defaults to FALSE.

Value

METASOFT output file is allocated to specified object

Examples

METASOFT_Output <- METASOFT_Run(Data = METASOFT_Example_Input)
