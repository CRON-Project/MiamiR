Munged, Covariate & Modelling Data

Description

Munged, Covariate & Modelling Data

Usage

Model_Munge(..., session = NULL)

Arguments

...

Shadow argument to detect arguments passed; defaults to empty list()

Model_Object

Base R model generated from running an lm() or glm(); defaults to NULL

Verbose

Prevent display of progress bar as function is running and instead show key milestone outputs/messages (mainly for debugging purposes); defaults to FALSE

Value

Munged dataframe amenable to plotting test statistics which can then be passed to other MiamiR functions

Examples

 Model_Munged <- Model_Munge(Model_Object = "Model_One")
