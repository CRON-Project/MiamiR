Tube_AlloysR Documentation

Visualise GWAS Data

Description

Visualise GWAS Data

Usage

Tube_Alloys(
  Data = NULL,
  Chromosomes = 1,
  Phenos = NULL,
  Covars = NULL,
  Phenos_Covars = NULL,
  Pheno_Name = NULL
)

Arguments

Data

These are the summary statistics (either file path(s), environmental object(s) or a variable(s) containing such contents) to be plotted; defaults to NULL

Chromosomes

Manually specify which chromosomes from the summary statistics to run the chromosome sensitive functions on; defaults to NULL, leading to selective default filtering

Phenos

Specify GWAS phenotype file, when Phenos_Covars is FALSE - used for GWAS model overview Forest_Plot(); defaults to NULL

Covars

Specify GWAS covariate file, when Phenos_Covars is FALSE - used for GWAS model overview Forest_Plot(); defaults to NULL

Phenos_Covars

Specify combined phenotype & covariate file, when Phenos & Covars are both FALSE - used for GWAS model overview Forest_Plot(); defaults to NULL

Pheno_Name

When Phenos_Covars is TRUE, specify the name of the column containing the GWAS Phenotype - used for GWAS model overview Forest_Plot(); defaults to NULL

Value

Images of Manhattan Plot(s), Regional Plots(s) and Forest Plot(s) are allocated to specified object and the resulting ggplot and grob object(s) can then be saved to an image

Examples

Plots <- Tube_Alloys(Data = Intelligence_Sum_Stats,
                               Phenos = Fake_PHENOS_Binary,
                               Covars = Fake_COVARS, Chromosomes = 1 )
