Fake_Demo_DataR Documentation

1,000 Participant Simulated Sample of Dementia Cases, with Covariate information for modelling

Description

A dataset containing simulated dementia cases and covariates for participants The variables are as follows:

Usage

data(Fake_Demo_Data)

Format

A data frame with 1,000 rows and 6 variables (simulated)

Details

  • Age = Age (in years)

  • Sex = Genetic Sex

  • BMI = Continuous body mass index

  • Ethnicity = Broad self-report ethnicity

  • Dementia = Diagnosis of dementia, with 1 representing a case

  • Location = Sample site, grouped by dwelling type
